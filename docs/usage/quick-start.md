# Quick Start

Create your first bento page in 5 minutes!

## Step 1: Open the Builder

Go to OpenBento and you'll see the visual editor with a 9Ã—9 grid.

## Step 2: Set Up Your Profile

In the sidebar, configure your profile:

1. **Name** - Your display name
2. **Bio** - A short description
3. **Avatar** - Upload a profile picture
   - Choose shape: circle, square, or rounded
   - Add border and shadow effects

## Step 3: Add Your First Block

Click **+ Add Block** in the sidebar and choose a block type:

- **Social** - Link to your social profiles
- **Link** - Add custom links
- **Text** - Add bio or notes

## Step 4: Position Your Block

- **Drag** the block to move it on the grid
- **Resize** by dragging the corners
- Blocks snap to the grid automatically

## Step 5: Customize Colors

Click on any block to edit:

1. Change background color
2. Adjust text color
3. Add gradients or custom backgrounds

## Step 6: Add More Blocks

Build your layout by adding more blocks:

- Mix different block types
- Create visual hierarchy with sizes
- Use spacer blocks for layout control

## Step 7: Preview

Click the **Preview** button to see how your bento looks on different devices.

## Step 8: Export

When you're happy with your design:

1. Click **Export**
2. Choose your deployment target
3. Download the complete project

That's it! You now have a deployable bento page.

## Tips

- **Start simple** - Begin with 3-4 blocks and expand
- **Use contrast** - Mix light and dark blocks
- **Balance** - Distribute blocks evenly across the grid
- **Mobile first** - Preview on mobile to ensure readability
