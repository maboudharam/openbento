# Block Types

OpenBento includes 7 block types to create your perfect bento layout.

## Social Block

Connect your social profiles with branded colors and icons.

**Features:**
- 26+ supported platforms
- Automatic URL generation from username
- Branded colors and icons
- Optional follower count display

**Supported Platforms:**

| Platform | Platform | Platform |
|----------|----------|----------|
| X (Twitter) | Instagram | TikTok |
| YouTube | GitHub | GitLab |
| LinkedIn | Facebook | Twitch |
| Dribbble | Medium | Dev.to |
| Reddit | Pinterest | Threads |
| Bluesky | Mastodon | Substack |
| Patreon | Ko-fi | Buy Me a Coffee |
| Snapchat | Discord | Telegram |
| WhatsApp | Spotify | Custom |

**How to use:**
1. Add a Social block
2. Select the platform
3. Enter your username/handle
4. The URL is generated automatically

## Link Block

Create clickable links with custom styling.

**Features:**
- Title and subtitle
- Custom URL
- Background colors and gradients
- Icon support

**Use cases:**
- Portfolio links
- Product pages
- Newsletter signups
- Any external URL

## Text Block

Add text content to your bento.

**Features:**
- Rich text display
- Custom fonts and sizes
- Background customization

**Use cases:**
- Bio sections
- Quotes
- Announcements
- Contact info

## Media Block

Display images and GIFs.

**Features:**
- Image upload
- Position control (focal point)
- Responsive sizing
- GIF support

**Supported formats:**
- JPEG / JPG
- PNG
- GIF
- WebP

## YouTube Block

Embed YouTube content in three modes.

### Single Mode
Display one video with thumbnail and play button.

### Grid Mode
Show multiple videos in a grid layout.

### List Mode
Display videos as a scrollable list with thumbnails.

**How to use:**
1. Add a YouTube block
2. Choose display mode
3. Enter video URL or channel ID
4. Videos load automatically

## Map Block

Embed an interactive Google Maps location.

**Features:**
- Search any location
- Interactive map display
- Automatic zoom level

**How to use:**
1. Add a Map block
2. Enter location name or address
3. Map updates automatically

## Spacer Block

Empty blocks for layout control.

**Use cases:**
- Create visual spacing
- Balance layouts
- Separate sections
- Fill grid gaps

## Block Sizing

All blocks can be resized on the 9×9 grid:

| Size | Best for |
|------|----------|
| 3×3 | Icons, small items |
| 3×6 | Vertical content |
| 6×3 | Horizontal content |
| 6×6 | Featured content |
| 9×3 | Full-width banners |

## Block Colors

Each block supports:

- **Solid colors** - Tailwind color palette
- **Custom colors** - Any hex value
- **Gradients** - Custom CSS gradients
- **Text colors** - Light or dark for contrast

## Tips

- **Hierarchy** - Use larger blocks for important content
- **Contrast** - Alternate light and dark backgrounds
- **Spacing** - Use spacer blocks to avoid clutter
- **Consistency** - Keep similar content in similar-sized blocks
