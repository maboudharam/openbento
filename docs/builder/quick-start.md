# Quick Start: Builder Setup

Get the OpenBento builder running in under 2 minutes.

## Prerequisites

- Node.js 18+

## Install & Run

```bash
# Clone
git clone https://github.com/yoanbernabeu/openbento.git
cd openbento

# Install
npm install

# Run
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — you're ready to create!
